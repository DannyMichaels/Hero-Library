Hero Library

Hero search made simple

Hero Library (developed by Daniel Michael), is a website where the user will be able to search and find Super-Hero data such as stats and images.
once the user enters his input the image of searched hero will spawn, with additional description and data underneath.
If the searched hero has a related movie, the related movie will appear underneath.

by the way, you can also search for villians!
